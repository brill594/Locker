# Locker 🌿

Locker 不仅仅是一个应用锁，它是一个帮助你暂时放下手机、回归当下的工具。

---

## ✨ 核心特性

* **极简设计**：无干扰的纯净界面，只显示最重要的倒计时，让视觉回归平静。
* **沉浸式锁定**：通过暂时接管桌面（Launcher），为你构建一个无法退出的专注空间。
* **静音守护**：自动开启勿扰模式并静音通知，让此刻只属于你。
* **智能还原**：倒计时结束后，自动带你回到原本熟悉的手机桌面，无需手动操作。
* **安全无痛**：基于 Shizuku 实现，无需 Root 权限，安全且对系统零侵入。

---

## 🚀 开始使用前的准备

Locker 需要配合 **Shizuku** 使用以获得必要的系统权限。

1.  请确保你的手机已安装并激活了 [Shizuku](https://shizuku.rikka.app/)。
2.  下载并安装 Locker。

---

## 📖 权限开启指南（重要）

为了实现完整的专注体验，Locker 需要**无障碍服务**权限来屏蔽干扰按键。

⚠️ **注意**：如果你是手动安装的 APK（而非从应用商店下载），Android 13 及以上系统可能会为了安全，默认“冻结”无障碍服务的授权。请按照以下步骤解开限制：

### 第一步：允许受限制的设置
1.  打开手机系统的 **「设置」** -> **「应用」** -> 找到 **「Locker」**。
2.  进入应用详情页（即显示卸载/强行停止的页面）。
3.  点击右上角的 **三个点 (⋮)** 菜单(如果不显示⋮：可尝试打开Locker，跟随无障碍激活步骤，到系统内触发受限弹窗，再到应用详情中即可出现⋮)。
4.  点击 **“允许受限制的设置” (Allow restricted settings)**。
    * *验证：此时系统可能会要求你验证指纹或密码。*

### 第二步：开启无障碍服务
1.  回到 Locker 应用内，或前往系统 **「设置」** -> **「无障碍」**。
2.  找到 **“Locker Focus Guard”**。
3.  此时开关应已变为可用状态，点击开启即可。

---

## 🎮 如何使用

1.  **连接服务**：打开 App，点击“授权 Shizuku”按钮，确保服务已连接。
2.  **设定时长**：滑动时间滚轮，选择你希望专注的时长（1分钟 - 120分钟）。
3.  **开始专注**：点击 **START**。
    * *此时手机将进入锁定模式，通知将被屏蔽，返回键将被拦截。*
    * *请放下手机，享受这段专注时光。*
4.  **自动结束**：倒计时归零后，Locker 会自动解锁并带你回原本的桌面。

---

## 🔒 隐私与安全

* **无联网权限**：Locker 不需要网络权限，不会上传你的任何数据。
* **无 Root 风险**：所有高级功能均通过 Shizuku 桥接，不破坏系统分区，卸载即还原（默认主页面程序需要手动还原）。
---

## 📝 开发说明

本项目使用 Kotlin 与 Jetpack Compose 构建，利用 Shizuku API 实现无 Root 环境下的系统级管控。

* **Architecture**: MVVM
* **Tech Stack**: Jetpack Compose, Coroutines, Shizuku API
* **Compatibility**: Android 11+ (Recommended Android 14/15)
